## 🚀 Live Links

* 🌐 Frontend:https://investra-e-commerce.vercel.app/
* 🛠 Admin Panel:https://investra-e-commerce-6m6y.vercel.app/
* ⚙️ Backend API: https://investra-e-commerce.onrender.com

---

## 📌 Project Overview

investra is a full-stack e-commerce web application where users can browse products, create accounts, and make purchases, while admins can manage products and users through a dedicated dashboard.

---

## ✨ Features

### 👤 User Features

* User Signup & Login (Authentication)
* Browse Products
* Add to Cart
* Place Orders
* Responsive UI

### 🛠 Admin Features

* Admin Login
* Add / Update / Delete Products
* Manage Orders
* Dashboard Access

---

## 🧑‍💻 Tech Stack

### Frontend

* React.js / Vite
* HTML, CSS, JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB (Cloud using MongoDB Atlas)

### Deployment

* Frontend & Admin: Vercel
* Backend: Render

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

---

### 2️⃣ Backend Setup

```bash
cd backend
npm install
```

Create `.env` file:

```
MONGO_URI=your_mongodb_connection_string
PORT=4000
```

Run backend:

```bash
node index.js
```

---

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

### 4️⃣ Admin Panel Setup

```bash
cd admin
npm install
npm run dev
```
